# ABSOLUTE Studio

Сайт студии профессиональной видеосъёмки в Перми.

## Технологии

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS v4
- **Animation:** Motion (framer-motion)
- **Icons:** Phosphor Icons
- **Fonts:** Geist Sans + Geist Mono

## Возможности

- 3 языка: русский, английский, китайский
- Плавные scroll-анимации и hover-эффекты
- Адаптивный дизайн (mobile-first)
- Модальное окно просмотра видео
- Форма обратной связи
- SEO-оптимизация (meta tags, OG, hreflang, sitemap)
- Security headers (CSP, X-Frame-Options)

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
npm start
```
